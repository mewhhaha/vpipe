# Changelog

Notable changes to the buildable `vpipe` example programs are recorded here.

## 0.1.0.0 — unreleased

- Added runnable triangle, indexed textured/depth cube, 100,000-particle
  compute-to-graphics simulation, two-pass offscreen vignette, headless PNG,
  and shadertoy programs.
- Added deterministic screenshot modes, reviewed 64×64 goldens, explicit
  per-channel comparison tolerance, and a regeneration command.
